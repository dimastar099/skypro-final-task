Git

  • git init — инициализация нового репозитория
  • git status — показать текущие изменения в проекте
  • git add — добавить файл или файлы в индекс (подготовить к коммиту)
  • git commit — зафиксировать изменения
  • git push — отправить изменения в удалённый репозиторий
  • git pull — получить последние изменения из удалённого репозитория
  • git branch — показать список веток
  • git checkout — переключиться на другую ветку
